A WordPress plugin to provide a custom image layout tool with canvas functionality. Upload, position, resize, rotate, and export your image layout with ease.

== Description ==

Custom Image Layout Tool allows you to:
- Upload multiple images
- Resize images with slider or dimension input
- Rotate and duplicate images
- Snap to grid for better alignment
- Export layout as JPG or PNG
- Select from different sheet sizes
- View real-time image dimensions
- Intuitive and responsive interface

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin from your WordPress admin dashboard.
3. Insert the shortcode `[custom_image_layout]` into any post or page.

== Frequently Asked Questions ==

= How do I use the plugin? =  
Just use the `[custom_image_layout]` shortcode anywhere you want to display the tool.

= Can I export my layout? =  
Yes, you can export your final layout as a JPG or PNG image.

= Can I set image size manually? =  
Yes, you can either use quick width buttons or manually enter width and height in cm.
